This directory contains the drivers for the SafeNet Sentinel SuperPro dongle
used by the Green Hills tools when dongle-locked licenses are used.

For Windows versions before Windows 7, use the following driver installer:
    windows/Sentinel Protection Installer 7.5.0.exe

For Windows 7, use the following driver installer:
    windows/Sentinel Protection Installer 7.6.3.exe

If you experience problems with your dongle, please check the SafeNet web site
for updated Sentinel SuperPro drivers:

http://www.safenet-inc.com/

Please contact Green Hills technical support if you have any questions about
supported platforms or any problems using your dongle-locked licenses with
the Green Hills tools (see http://support.ghs.com/contact.html).
